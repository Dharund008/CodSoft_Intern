Blog Application

Mern Stack

The MERN Blog Application is a full-stack web application designed to facilitate the creation, management, and interaction with blog posts. Users can like and unlike posts created by others, leave comments, and utilize a user and admin dashboard. The application also provides a WYSIWYG (What You See Is What You Get) editor and enables users to upload images, which are stored in Cloudinary. The system is based on the MERN stack, with MongoDB used to store user credentials and data.

Movie Streaming Platform

Mern Stack

The Movie Streaming Application is a web-based streaming platform designed to replicate the essential features of the Netflix service. It aims to provide users with the ability to browse, search, and stream a wide variety of movies and TV shows. The application will include user authentication, content categorization, personalized recommendations, and a user-friendly interface. The system will be developed using the MERN stack.

Women Safety App

Kavach Hackathon

Free Bird is a modern women's safety app offering a range of features like SMS alerts, emergency calling, voice triggering, shake-to-safety, and live location sharing. It goes a step further with advanced functionalities such as offline location sharing and a panic mode with sound detection. What makes Free Bird unique is its special algorithm, utilizing multi-model data (audio, video, image, motion detection) to assess the user's situation and initiate appropriate actions. With Free Bird, your safety is paramount, ensuring peace of mind in every circumstance.
